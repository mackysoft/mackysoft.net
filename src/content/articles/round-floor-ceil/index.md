「RoundとFloorとCeil、どれがどれだっけな～」みたいなことになりがちなので、備忘録的にまとめました。

| 関数 | 説明 |
| --- | --- |
| [Round](https://docs.unity3d.com/ScriptReference/Mathf.Round.html) / [RoundToInt](https://docs.unity3d.com/ScriptReference/Mathf.RoundToInt.html) | 四捨五入 |
| [Floor](https://docs.unity3d.com/ScriptReference/Mathf.Floor.html) / [FloorToInt](https://docs.unity3d.com/ScriptReference/Mathf.FloorToInt.html) | 切り捨て（「床」という意味なので、**下げる**という風に覚えられます） |
| [Ceil](https://docs.unity3d.com/ScriptReference/Mathf.Ceil.html) / [CeilToInt](https://docs.unity3d.com/ScriptReference/Mathf.CeilToInt.html) | 切り上げ（「天井」という意味なので、**上げる**という風に覚えられます） |
