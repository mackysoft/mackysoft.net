이 사이트는 게임, 에셋, 기술 글을 공개하는 개인 사이트입니다. 이 페이지에서는 사이트에서 실제로 사용하고 있는 기능에 맞춰 정보의 수집과 이용 방식을 설명합니다.

## 운영자 정보와 연락처

이 사이트의 운영자는 Hiroya Aramaki(아라마키 히로야) / Makihiro입니다.

문의는 [연락처](/contact/) 페이지에서 안내하는 방법으로 받습니다. GitHub를 통해 받은 연락도 이 방침에 따라 처리합니다.

## 수집하는 정보

이 사이트는 페이지 표시, 언어 전환, 이용 분석, 외부 서비스 연동을 위해 사이트 자체 또는 사이트가 사용하는 외부 서비스에서 다음 정보를 처리할 수 있습니다.

- 쿠키 기반 식별자, IP 주소, 브라우저 및 기기 정보, 유입 경로, 방문한 URL, 접속 시각, 화면 해상도, 대략적인 위치 정보와 같은 기술 정보
- 초기 표시 언어를 결정하는 데 사용하는 `navigator.languages` 같은 브라우저 언어 설정
- 테마와 언어 설정을 유지하기 위해 `localStorage`에 저장하는 `mackysoft-theme`와 `mackysoft-locale`
- 언어 전환 후 스크롤 위치를 복원하기 위해 `sessionStorage`에 저장하는 `mackysoft-locale-scroll`
- 사이트 내 검색 측정을 완료하기 위해 `sessionStorage`의 `__pending_site_search__`에 임시 저장하는 검색어와 분석 파라미터

이메일 또는 GitHub를 통해 운영자에게 연락하면, 사이트는 이름, 이메일 주소, 계정명, 제목, 본문, 그리고 사용자가 직접 보내는 기타 정보를 받을 수 있습니다.

## 이용 목적

수집한 정보는 사이트 제공과 운영, 테마 및 언어 설정 유지, 이용 현황 분석, 편의성 개선, 악용이나 기술적 문제 대응, 문의 처리에 사용합니다.

수집한 정보를 광고 기능이나 계정 관리 용도로는 사용하지 않습니다.

## 분석

이 사이트는 Google Analytics 4를 사용합니다. Google Analytics 4는 Google tag, `_ga`와 `_ga_<container-id>` 같은 쿠키 기반 식별자, 그리고 관련 기술 데이터를 사용해 이용 정보를 수집하며, Google이 이를 처리할 수 있습니다.

이 사이트는 기본적인 페이지뷰 측정 외에도 사이트 내 검색 제출, 테마 전환, 언어 전환, 외부 링크 클릭을 측정합니다. 이 과정에서 페이지 URL, 유입 URL, 페이지 제목, 화면 해상도, 사이트 내 검색에서 제출된 검색어, UI 위치, 클릭한 라벨, 대상 URL, 전환 대상 테마, 전환 대상 언어 같은 정보가 Google Analytics 4로 전송될 수 있습니다.

Google signals와 광고 개인화 기능은 사용하지 않습니다.

- [Google Analytics Privacy Disclosures Policy](https://support.google.com/analytics/answer/7318509?hl=en)
- [GA4 데이터 수집](https://support.google.com/analytics/answer/11593727?hl=en)
- [Google 개인정보처리방침](https://policies.google.com/privacy)

## 외부 서비스

이 사이트는 Cloudflare Workers에서 제공됩니다. 사이트에 접속하면 전송, 보안, 가용성 확보를 위해 Cloudflare가 IP 주소, 요청 정보, 응답 정보 등을 처리할 수 있습니다.

이 사이트는 Google Fonts를 사용해 글꼴을 제공합니다. 글꼴을 불러올 때 사용자의 브라우저와 Google 서버 간 통신이 발생합니다.

이 사이트는 GitHub의 공개 저장소 정보와 릴리스 정보를 바탕으로, 빌드 시 설명문과 커버 이미지를 생성해 반영할 수 있습니다. 현재 공개 페이지의 구현에서는 일반적인 페이지 열람 시 사용자의 브라우저가 GitHub 관련 서버에서 이미지를 직접 가져가도록 되어 있지 않습니다.

게임 페이지의 동영상 삽입에는 YouTube의 개인정보 보호 강화 모드인 `youtube-nocookie.com`을 사용합니다. 삽입된 플레이어를 표시하거나 동영상을 재생하면 YouTube와 관련된 통신이 발생할 수 있습니다.

GitHub, X, Zenn 같은 외부 사이트로 이동한 뒤에는 각 서비스 사업자가 정한 개인정보처리방침 등이 적용됩니다.

## 해외 사업자에 의한 처리

이 사이트는 Google, GitHub, Cloudflare 등 일본 외 지역에 있는 사업자의 서비스를 사용합니다. 이러한 서비스를 통해 다뤄지는 정보는 일본 외 지역에서 처리될 수 있습니다.

## 이용자의 통제 방법

쿠키, `localStorage`, `sessionStorage`의 저장은 사용자가 브라우저 설정에서 삭제하거나 비활성화할 수 있습니다. 다만 이 경우 테마와 언어 설정 유지, 언어 전환 후 화면 복원, 일부 측정 기능이 제대로 동작하지 않을 수 있습니다.

Google Analytics에 의한 정보 수집을 중지하려면 Google이 제공하는 옵트아웃 애드온 등의 도구를 사용해 주세요.

- [Google Analytics Opt-out Browser Add-on](https://tools.google.com/dlpage/gaoptout/intl/en/success.html)
