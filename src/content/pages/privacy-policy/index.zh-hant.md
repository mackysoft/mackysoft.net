本網站是用來發佈遊戲、素材與技術文章的個人網站。本頁會依照本網站實際使用的功能，說明資訊蒐集與使用的方式。

## 營運者資訊與聯絡方式

本網站的營運者為 Hiroya Aramaki（荒牧裕也）/ Makihiro。

如需聯絡，請使用 [聯絡方式](/contact/) 頁面所介紹的方法。透過 GitHub 收到的聯絡，也會依照本政策處理。

## 蒐集的資訊

本網站為了顯示頁面、切換語言、進行流量分析，以及與外部服務整合，本網站本身或本網站使用的外部服務可能會處理以下資訊。

- Cookie 等識別碼、IP 位址、瀏覽器與裝置資訊、來源頁面、瀏覽的 URL、瀏覽時間、螢幕解析度、概略位置資訊等技術資訊
- `navigator.languages` 等用於判定顯示語言的瀏覽器語言設定
- 為了保存主題設定與顯示語言，儲存在 `localStorage` 中的 `mackysoft-theme` 與 `mackysoft-locale`
- 為了在切換語言時還原捲動位置，儲存在 `sessionStorage` 中的 `mackysoft-locale-scroll`
- 為了在從頁首搜尋移動到搜尋結果頁後，只在同一個分頁內完成一次搜尋送出計測而暫時儲存在 `sessionStorage` 的值

如果你透過電子郵件或 GitHub 聯絡本站，本站會取得你的姓名、電子郵件地址、帳號名稱、主旨、本文，以及其他由使用者自行提供的資訊。

## 使用目的

蒐集到的資訊將用於網站的顯示與營運、保存主題設定與顯示語言、分析使用狀況、提升便利性、處理不當使用與技術故障，以及回覆詢問。

蒐集到的資訊不會用於廣告用途的擴充功能或會員管理。

## 流量分析

本網站使用 Google Analytics 4。Google Analytics 4 會使用 Google tag、Cookie 等識別碼（例如 `_ga`、`_ga_<container-id>`）以及相關技術資訊來蒐集使用狀況，而這些資料可能會由 Google 處理。

本網站除了頁面瀏覽的基本流量計測之外，也會計測站內搜尋的送出、主題切換、語言切換，以及外部連結的點擊。因此，頁面 URL、來源 URL、頁面標題、螢幕解析度、在站內搜尋中送出的搜尋詞、畫面上的位置、被點擊目標的標籤、跳轉目標 URL、切換後的主題、切換後的語言等資訊，都有可能被送往 Google Analytics 4。

本網站不使用 Google signals 或用於廣告的個人化功能。

- [Google Analytics Privacy Disclosures Policy](https://support.google.com/analytics/answer/7318509?hl=en)
- [GA4 的資料蒐集](https://support.google.com/analytics/answer/11593727?hl=en)
- [Google Privacy Policy](https://policies.google.com/privacy)

## 外部服務

本網站部署於 Cloudflare Workers 上。當你造訪本站時，Cloudflare 可能會基於內容傳遞、安全性與可用性的需求，處理 IP 位址、請求資訊與回應資訊等資料。

本網站使用 Google Fonts 提供字型。載入字型時，使用者的瀏覽器會與 Google 的伺服器通訊。

本網站可能會根據 GitHub 的公開儲存庫資訊與 Release 資訊，在建置時生成並反映說明文字與封面圖片。就目前公開頁面的實作而言，一般瀏覽頁面時，使用者的瀏覽器不會因為顯示這些內容而直接向 GitHub 相關伺服器請求圖片。

遊戲頁面的影片嵌入使用了 YouTube 隱私權加強模式下的 `youtube-nocookie.com`。顯示嵌入播放器或播放影片時，仍可能會發生與 YouTube 相關的通訊。

在移動到 GitHub、X、Zenn 等外部網站後，將適用各該服務業者所制定的隱私權政策等規定。

## 由海外業者進行的處理

本網站使用 Google、GitHub、Cloudflare 等位於日本以外之業者所提供的服務。透過這些服務所處理的資訊，可能會在日本境外被處理。

## 使用者可採取的控制方式

Cookie、`localStorage`、`sessionStorage` 的儲存，可由使用者透過瀏覽器設定刪除或停用。不過在這種情況下，主題設定與顯示語言的保存、切換語言時的畫面還原，以及部分搜尋送出計測可能無法正常運作。

如果你希望停止 Google Analytics 的資訊蒐集，請使用 Google 提供的停用外掛程式等工具。

- [Google Analytics Opt-out Browser Add-on](https://tools.google.com/dlpage/gaoptout/intl/en/success.html)
