# mackysoft.net

## Prerequisites

- Node.js 22.12.0 以上 23 未満
- npm 10 以上

## Setup

```bash
npm install
```

## Available Commands

```bash
npm run dev
npm run build
npm run preview
npm run check
npm run import:wordpress
```

## WordPress Import

旧 WordPress ブログ記事をローカル原本へ移すときは、次を実行します。

```bash
npm run import:wordpress
```
